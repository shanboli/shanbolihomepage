Simple Life 2.0 - A Free CSS Template

Simple life was originally designed by Pat Heard (Fullahead.org). It is a beauitully designed and coded template. TemplateFusion has come in and modified minor things throughout the template and is releasing it to the general public. (Including, background image, highlight tag, forms, lists, greenbg and more). This template is released under the Creative Commons Attribution 2.5 canada License. This means you may use it for a commercial or personal site, with no limitation, except that you keep the link in your footer to TemplateFusion and FullAhead. The wood background was created by All Things Designed. Kudos for creating such a beautiful tiling image.

Introduction

A simple, XHTML 1.0 tableless template without a default sidebar. You can add one if you want using the custom CSS width classes, but sometimes, all you really need is one column.

The template displays properly in all the usual browsers, with an added filter hack to get .png opacity working in IE. If you look in the html source near the top, you'll see a conditional comment that loads a style sheet for IE 5.0 and up. You can get rid of this, and everything will still work, except you'll loose the purty content dropshadow.
